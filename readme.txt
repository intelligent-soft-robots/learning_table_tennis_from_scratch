# Instructions

- install the treep project "PAM_MUJOCO" following these [instructions](https://github.com/intelligent-soft-robots/intelligent-soft-robots.github.io/wiki).

- if running in real time :
    - start the executable start_robots
    - in another window, python run ```python hysr_one_ball_random.py``` or ```python hysr_one_ball_random.py```

- if running in accelerated time:
    - start the executable start_robots_accelerated
    - in another window, python run ```python hysr_one_ball_random.py accelerated``` or ```python hysr_one_ball_random.py accelerated```
